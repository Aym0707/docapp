# سیستم رزرو وقت ملاقات - کلینیک سلامتی کابل

این یک سیستم رزرو وقت ملاقات امن برای کلینیک‌های درمانی در افغانستان است که به زبان فارسی (گویش افغانستان) توسعه یافته است.

## ویژگی‌ها

- فرم ثبت نام بیمار به زبان فارسی افغانستان
- ذخیره‌سازی امن اطلاعات در Google Sheets
- طراحی واکنش‌گرا برای همه دستگاه‌ها
- امنیت بالا با رمزگذاری و اعتبارسنجی
- نمایش تاریخ شمسی (هجری خورشیدی)
- تولید شماره رهگیری خودکار

## پیش‌نیازها

1. یک حساب Google برای دسترسی به Google Sheets
2. یک حساب Vercel برای استقرار ( deployment)
3. دسترسی به اینترنت برای بارگذاری فایل‌ها

## راهنمای استقرار در Vercel

### مرحله ۱: آماده‌سازی Google Sheets

1. به [Google Sheets](https://sheets.google.com) بروید و یک صفحه‌گسترده جدید ایجاد کنید
2. نام صفحه‌گسترده را به "Appointment System" تغییر دهید
3. ID صفحه‌گسترده را از URL کپی کنید (مثال: `https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit`)
4. به [Google Cloud Console](https://console.cloud.google.com) بروید
5. یک پروژه جدید ایجاد کنید یا از پروژه موجود استفاده کنید
6. به "APIs & Services" > "Credentials" بروید
7. روی "Create Credentials" کلیک کرده و "Service Account" را انتخاب کنید
8. نام سرویس اکانت را وارد کنید و روی "Create" کلیک کنید
9. نقش "Editor" را به سرویس اکانت اختصاص دهید
10. روی "Done" کلیک کنید
11. به سرویس اکانت ایجاد شده بروید و روی "Keys" کلیک کنید
12. "Add Key" > "Create new key" را انتخاب و فرمت JSON را انتخاب کنید
13. کلید JSON دانلود خواهد شد - این فایل را ایمن نگه دارید

### مرحله ۲: آماده‌سازی محیط Vercel

1. به [Vercel](https://vercel.com) بروید و وارد حساب خود شوید
2. روی "Add New Project" کلیک کنید
3. مخزن Git خود را انتخاب کنید یا از طریق دستی آپلود کنید
4. تنظیمات پروژه را پیکربندی کنید:
   - Framework Preset: "Other"
   - Build Command: خالی بگذارید
   - Output Directory: `.`
   - Install Command: `npm install` (اگر از package.json استفاده می‌کنید)

### مرحله ۳: تنظیم متغیرهای محیطی در Vercel

در تنظیمات پروژه Vercel، به بخش "Environment Variables" بروید و متغیرهای زیر را اضافه کنید:

1. `GOOGLE_SHEET_ID`: ID صفحه‌گسترده شما (از مرحله ۱)
2. `GOOGLE_SERVICE_ACCOUNT_CREDENTIALS`: محتوای فایل JSON کلید سرویس اکانت (کل فایل JSON را کپی کنید)

### مرحله ۴: استقرار پروژه

1. فایل‌های پروژه را در Vercel آپلود کنید یا از Git استفاده کنید
2. روی "Deploy" کلیک کنید
3. پس از استقرار موفق، URL پروژه شما در دسترس خواهد بود

## تست سیستم

1. به URL استقرار شده در Vercel بروید
2. فرم را با اطلاعات تست پر کنید
3. روی "ارسال درخواست ملاقات" کلیک کنید
4. پیام موفقیت را بررسی کنید
5. به Google Sheets خود بروید و تأیید کنید که داده‌ها ذخیره شده‌اند

## امنیت

- تمام ارتباطات با پروتکل HTTPS رمزگذاری می‌شوند
- اطلاعات احراز هویت Google در متغیرهای محیطی ذخیره می‌شوند
- اعتبارسنجی ورودی در سمت کلاینت و سرور انجام می‌شود
- اطلاعات بیماران فقط برای مقاصد درمانی استفاده می‌شود

## توسعه بیشتر

- افزودن سیستم SMS برای ارسال تأییدیه
- ایجاد پنل مدیریت برای مشاهده رزروها
- افزودن سیستم یادآوری قرار ملاقات
- یکپارچه‌سازی با تقویم Google
